# Test Case Generation API Console

React + Vite scaffold for the page design generated from the PRD in [docs/test_case_generation_api_v1.0_UED](/C:/Users/iflytek/.codex/worktrees/a53d/code/codex/docs/test_case_generation_api_v1.0_UED).

## Commands

- `npm install` to install dependencies
- `npm run dev` to start the Vite dev server
- `npm run build` to generate a production bundle
- `npm run preview` to preview the production bundle locally

## Structure

- `src/` React application source
- `docs/test_case_generation_api_v1.0_UED/` original HTML design demo and UED notes

## Notes

- The React scaffold reuses the UED design tokens from `docs/test_case_generation_api_v1.0_UED/assets/styles.css`.
- The current pages use mock data and are ready to be wired to real APIs.
