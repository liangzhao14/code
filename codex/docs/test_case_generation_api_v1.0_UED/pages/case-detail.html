CASE-001 / 任务完成后获取结果成功

满足 PRD 中“单条用例详情需便于复制和二次编辑”的要求，采用清晰的描述块和操作按钮布局。

P0 正常

用例信息

返回列表
所属模块
结果获取
场景描述
任务成功后，调用结果接口获取完整结构化测试用例集合。
前置条件
任务状态为成功，结果未过期，调用方具备归属权限。
标签
结果获取 / JSON / 权限校验

测试步骤

1. 调用结果接口，传入合法任务 ID 与应用签名。
2. 校验响应状态码为 200，返回体包含 resultMeta、summary 和 cases。
3. 校验 cases 中每条用例含标题、前置条件、步骤、预期结果、优先级。
4. 校验返回体展示生成时间、模板信息与覆盖维度说明。

预期结果

接口返回完整的结构化结果，字段齐全，任务归属合法，未出现“任务尚未完成”或“结果已过期”类错误信息。
